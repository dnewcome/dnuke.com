# VirtualBox pause feature

I just found that [VirtualBox](http://www.virtualbox.org/) has the ability to instantly pause a running VM. Under VMWare server you had to suspend the instance if you wanted to free up your CPU. I think that Microsoft Virtual PC may have this instant pause feature as well. This is pretty cool for us laptop users that are on battery a lot.  I have a lot of ram, so I don’t really mind `parking’ an instance and time (not to mention battery life) is precious.  It doesn’t sound like a big deal, but waiting several minutes to fire up a VM instance that you just need for a second is a real workflow killer.
