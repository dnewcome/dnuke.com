# List of open source software used in the Pre

I was just poking around in the filesystem of the Palm Pre, and I found a [pdf file](https://newcome.wordpress.com/wp-content/uploads/2009/06/open-source-information.pdf) containing all of the open source software licenses used in the phone.  I thought it was interesting to see all of the different packages that went into making WebOS tick.
