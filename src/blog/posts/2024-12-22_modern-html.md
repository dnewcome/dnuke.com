# Modern HTML

I was looking around recently to see what the state of HTML5 was. I don’t even know if HTML5 is a buzzword anymore. I remember back in the day when we were trying to “fix” html and make the Web ok again. We always championed the idea of semantic markup and clean CSS. These were reactions to the bad old days of browser-specific hacks. But then we went off the deep end making the browser a full-on operating system.

This led me to wonder where are we today in a post-jquery world. How hard is it to make some columns that have draggable components?

I was pleasantly surprised to find out about [https://web.dev/articles/drag-and-drop](https://web.dev/articles/drag-and-drop) . The browser has a “draggable” property now. Wowzers.

This along with flexbox is pretty cool.
